# Lobox DropDown

## Lobox technical interview for the front-end developer position

### React + TypeScript + Vite

**Install dependencies**

```bash
$ pnpm install
```

**Run Unit Tests**

```bash
$ npx vitest run
```

**Start Dev Server**

```bash
$ pnpm dev
```

**Run E2E Tests**

```bash
$ npx playwright test
```
